# LinkedIn L&D Content Studio

An AI-powered **Streamlit web application** that helps **Learning & Development (L&D) professionals** generate high-quality LinkedIn posts from a simple topic idea.

Instead of manually researching, summarizing, and writing a post, the app uses a **3-step AI pipeline** to automatically transform a topic into a **ready-to-publish LinkedIn post**.

---

# Overview

Many professionals want to share insights on LinkedIn but struggle with:

* Finding reliable information
* Summarizing research quickly
* Writing engaging LinkedIn posts

This project solves that problem using an AI pipeline that performs:

1. **Topic Research**
2. **Research Summarization**
3. **LinkedIn Post Generation**

The result is a **structured research document, concise bullet summary, and a polished LinkedIn post**.

# Features

## AI Pipeline

The application uses a **three-step AI pipeline**:

### 1. Research Step

Generates structured research with sections such as:

* Key trends
* Important statistics
* Real-world examples
* Opportunities
* Risks or limitations

### 2. Summary Step

Converts research into **6–8 concise bullet points**.

Optional feature:

* Include a **counterpoint or limitation**.

### 3. LinkedIn Post Generation

Creates a **LinkedIn-style post** with:

* Strong hook
* Professional tone
* Practical insight
* Closing question
* Relevant hashtags

Post length is automatically targeted between **120 and 180 words**.

# UX Features

## Loading Indicators

The app uses Streamlit progress feedback such as:

* `Running research...`
* `Creating summary...`
* `Writing LinkedIn post...`

These are displayed using **Streamlit spinners**.

---

## Session State

The application uses **Streamlit session state** to persist data during interaction.

Stored values include:

```
topic
research_text
summary_text
post_text
history
```

This allows users to:

* Run steps independently
* Modify research before summarizing
* Edit summary before generating the post

---

## History Library

The sidebar includes a **mini content library**.

It stores the **last five runs**, including:

* Topic
* Timestamp
* Research output
* Summary output
* LinkedIn post

Clicking a topic automatically **reloads the content into the interface**.

---

# Validation Rules

To maintain proper workflow, the following checks are enforced:

### Topic Validation

The topic input cannot be empty.

### Summary Dependency

The summary step cannot run unless **research exists**.

### Post Dependency

The post generation step cannot run unless **summary exists**.

### Word Count Validation

The LinkedIn post must be between **120 and 180 words**.

If outside the range, a warning message appears.

# Installation

## 1. Clone the Repository

```
git clone https://github.com/your-username/linkedin-ld-content-studio.git
cd linkedin-ld-content-studio
```

---

## 2. Create a Virtual Environment

### Windows

```
python -m venv venv
venv\Scripts\activate
```

### macOS / Linux

```
python3 -m venv venv
source venv/bin/activate
```

---

## 3. Install Dependencies

```
pip install -r requirements.txt
```

---

# Setting the OpenAI API Key

This project requires an **OpenAI API key**.

## Windows (PowerShell)

```
setx OPENAI_API_KEY "your_api_key_here"
```

Restart the terminal after setting the variable.

---

## macOS / Linux

```
export OPENAI_API_KEY="your_api_key_here"
```

---

# Running the Application

Start the Streamlit server:

```
streamlit run app.py
```

The application will open automatically in your browser.

Default address:

```
http://localhost:8501
```
